# PathLens — Visual Query Builder

A highly interactive visual query builder built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Zustand** — state management
- **Vitest + React Testing Library** — testing
- **DnD Kit** — drag and drop
- **Shadcn/UI** — component library

## Getting Started

```bash
pnpm install
pnpm dev
```
